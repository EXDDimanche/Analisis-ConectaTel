# Analisis-ConnectaTel
Se evalúa el comportamiento de los clientes hasta el año 2024. Identificando patrones de uso

##Objetivo del Proyecto:
- El objetivo es poder identificar los patrones que siguen los clientes en base a las llamadas que hacen y los mensajes que mandan. Segmentandolos segun su edad y nivel de uso

##Datasets utilizados:
- Se utilizan 3 datasets:
    - plans, users_latam y usage

##Etapas del Análisis realizadas:
- Identificación de problemas de calidad
- Limpieza básica
- Summary Statistics
- Visualización y outliers
- Segmentacion
- Insight Ejecutivo
